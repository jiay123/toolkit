01

ShareX — 截图+录屏全能王

比微信截图好用10倍的免费截图工具，还能截图后自动上传到图床，链接一键复制。

📥 下载

浏览器访问：

https://getsharex.com

点 "Download" 下载安装包。仅支持 Windows 10/11。

⚙️ 安装步骤

  • 双击 ShareX-*-setup.exe
  • 语言选 "简体中文" → 点"确定"
  • 一路点 "下一步" → "我接受" → "安装"
  • 等待进度条跑完，点 "完成"

安装后 ShareX 会自动在右下角启动。

🎮 常用快捷键

功能快捷键
区域截图Ctrl + Shift + S
窗口截图Ctrl + Shift + W
全屏截图Ctrl + Alt + F
录制 GIFCtrl + Shift + G
录制视频Ctrl + Shift + R

💡 小技巧：截图自动上传

  • 打开 ShareX → 左侧点 "目标"
  • 把 "上传截图后保存图片" 打开
  • 默认图床选 Imgur（免费，不用注册）

以后截完图，链接自动复制好了，直接粘贴就能发给别人。

Umi-OCR — 免费离线文字识别

图片上的文字一键提取出来，完全离线免费，不限次数。43万人在用。

📥 下载

浏览器访问 GitHub Releases 页面：

https://github.com/hiroi-sora/Umi-OCR/releases

下载 Umi-OCR_Paddle_vxxx.7z.exe（最新版）。

支持 Windows / Mac / Linux，推荐用 Windows 版。

⚙️ 安装步骤（免安装版）

  • 下载完成后，双击运行
  • 它会自动解压到当前文件夹（等几秒）
  • 打开解压出来的文件夹
  • 找到 Umi-OCR.exe，双击运行

搞定。不需要装任何东西，打开就能用。

🎮 基本使用

截图识别：

  • 打开 Umi-OCR → 点左边 "截图OCR"
  • 按 Ctrl + G 框选要识别的区域
  • 文字自动提取出来，点 "复制"

批量识别：

  • 点左边 "批量OCR"
  • 把图片拖进去（可一次性拖几十张）
  • 点 "开始识别"，一键导出为 TXT 或 Word

Stirling-PDF — 浏览器里的PDF工具箱

50多种PDF功能，合并、拆分、压缩、转Word全免费。GitHub 76,000⭐。

📥 下载

方法一：下载桌面版（推荐）

https://www.stirlingpdf.com

点 "Download" → 选 Windows (.exe) → 下载安装。

方法二：在线版（不用安装）

https://www.stirlingpdf.com/app

浏览器打开就能用。支持 Windows / Mac / Linux。

⚙️ 安装步骤（Windows桌面版）

  • 双击下载的 .exe
  • 选 "简体中文" → 点"确定"
  • 一路 "下一步" → "我接受" → "安装"
  • 完成后在开始菜单打开 Stirling-PDF

浏览器会自动弹出操作界面。

🎮 常用功能

你想做什么操作步骤
合并多个 PDF点"合并PDF" → 拖入文件 → 排序 → 点"合并" → 下载
PDF 转 Word点"PDF转Word" → 上传 → 点"转换" → 下载
压缩 PDF点"压缩PDF" → 上传 → 选压缩比例 → 下载
加水印点"添加水印" → 输入文字 → 调位置 → 生成

所有操作在你自己的电脑上完成，文件不上传第三方服务器。

04

PowerToys — 微软官方Windows效率神器

微软自己出的免费工具合集，窗口分屏、批量重命名、取色器……装了电脑好用一倍。

📥 下载

方法一：微软商店安装（最推荐）

  • 按 Win 键，输入 "Microsoft Store"，回车
  • 搜索 "PowerToys"，点 "获取"

方法二：官网下载

https://github.com/microsoft/PowerToys/releases

下载 PowerToysSetup-xxx-x64.msi，双击安装。

仅支持 Windows 10/11。

⚙️ 安装步骤（微软商店版）

  • 打开 Microsoft Store
  • 搜索 PowerToys
  • 点 "获取" → 等它下载完
  • 打开 PowerToys，它会常驻右下角任务栏

🎮 最有用的几个功能

① FancyZones（窗口分屏）

  • 打开 PowerToys → 左边点 "FancyZones"
  • 点 "启动布局编辑器" → 选一个布局（比如三栏）
  • 按住 Shift 把窗口拖到对应区域，自动对齐

② PowerRename（批量重命名）

  • 选中一堆文件 → 右键 → "使用 PowerRename 重命名"
  • 输入要替换的文字，点"应用"

③ Text Extractor（截图识字）

  • 按 Ctrl + Shift + T，框选文字区域
  • 文字自动复制到剪贴板

④ ColorPicker（取色器）

  • 按 Ctrl + Shift + C，点屏幕上的颜色
  • 色值自动复制到剪贴板

OBS Studio — 专业录屏+直播，完全免费

B站UP主都在用的录屏软件，免费无水印无时长限制。

📥 下载

https://obsproject.com

点 "Download" → 选你的系统版本。支持 Windows / Mac / Linux。

⚙️ 安装步骤（Windows）

  • 双击 OBS-Studio-*-Full-Installer-x64.exe
  • 语言选 "简体中文" → 点"下一步"
  • 点 "我同意" → "下一步"
  • 安装位置保持默认 → "下一步" → "安装"
  • 等1-2分钟 → 点 "完成"

🎮 第一次使用设置

  • 选 "只优化录制，不要直播"
  • 分辨率选 1920x1080，帧率选 30 或 60
  • 点 "应用设置"

开始录屏：

  • 左下角 "来源" → 点 "+" → 选 "显示屏采集" → 确定
  • 点右边 "开始录制" → 录完点 "停止录制"

修改保存位置： 顶部菜单 → "文件" → "设置" → "输出" → "录像路径"

Open Design — 不会PS也能出设计图

打字就能生成海报、封面、配图。GitHub 62,600⭐，完全免费。

📥 下载

https://open-design.ai

点 "下载桌面版" → 选 Windows 版 (.exe) 或 Mac 版 (.dmg)。

支持 Windows / Mac。

⚙️ 安装步骤（Windows）

  • 双击 Open-Design-Setup-*.exe
  • 一路 "下一步" → "安装" → "完成"
  • 桌面上出现 Open Design 图标，双击打开
  • 点 "登录"（用 Google 账号或注册一个）
  • 登录后选 "设计" 模式

🎮 怎么用

做公众号封面：

  • 输入框打字，比如："帮我做一个咖啡店开业海报，深色风格，800x600"
  • AI 自动生成排版、配色、文字
  • 满意了直接截图保存

做小红书配图：

  • 说："给我做三张美妆推荐图，竖屏9:16"
  • AI 一次性生成三张
  • 点每张图 → "导出" → 保存到电脑

自带的AI模型已经配好了，不需要你单独买 API Key。
